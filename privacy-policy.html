We collect information that you provide directly to us and information that is automatically collected when you use our App.

Technical Data:

  • API request logs
  • Authentication tokens (stored securely)
  • Sync data timestamps
  • Connection information

2.3 Information Generated by AI Services

AI-Generated Content:

  • Journal entry scores (generated by AI analysis)
  • AI feedback and insights on your journal entries

5. Data Security

We implement industry-standard security measures to protect your information:

5.1 Technical Safeguards

  • Encryption: All sensitive data (journal entries, AI feedback) is encrypted using AES-256-GCM encryption
  • Secure Transmission: All data is transmitted over SSL/TLS encrypted connections
  • Password Security: Passwords are hashed using bcrypt and never stored in plain text
  • Authentication: We use JWT tokens with short expiration times and refresh token rotation
  • Access Controls: Strict access controls and authentication requirements

5.2 Operational Safeguards

  • Regular security audits and updates
  • Monitoring for suspicious activity
  • Rate limiting to prevent abuse
  • Secure database connections with IP restrictions

5.3 Data Storage

  • Your data is stored in secure, encrypted databases
  • Regular backups are performed
  • Data is stored in compliance with industry standards

Important: While we implement strong security measures, no method of transmission over the Internet or electronic storage is 100% secure. We cannot guarantee absolute security, but we strive to use commercially acceptable means to protect your information.
